Many channels via 3D heat structure

Purpose

See how well the code scales when using a 3D model of a core with many flow channels.

Setup

  • 3D heat structure for the core

  • 10 x 10 grid of channels going through the core

  • inlet and outlet pipes splitting the flow into individual core channels
